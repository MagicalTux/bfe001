12月29日〜1月4日の週。正月の静かな時期だが、新年早々いくつかの
地味な前進が着地。

**[echeck](https://github.com/KarpelesLab/echeck)** — 42コミット。
X.509 証明書に埋め込まれた Intel SGX クオート（OID
`1.3.6.1.4.1.311.105.1`）を取り出して検証するツールとライブラリ。
チェーン・オブ・トラストの完全検証まで行う。C ライブラリ（CLI
付き）と Go ライブラリの両方を提供。

**KLB スタックの Swift 移植。** 今週新しく2本：
**[swiftrest](https://github.com/KarpelesLab/swiftrest)**
（async/await と Swift actors で書いた REST クライアント。OAuth2 と
Ed25519 の API キー署名、チャンクドアップロード対応）と、
**[swiftbottle](https://github.com/BottleFmt/swiftbottle)**
（[Bottle](https://github.com/BottleFmt/bottle-id) 安全メッセージ
コンテナの Swift 実装。AES-256-GCM の多受信者暗号、Ed25519 /
ECDSA / RSA の複数署名、CBOR と JSON、[gobottle](https://github.com/BottleFmt/gobottle)
や pybottle と相互運用可能）。

**[qemuctl](https://github.com/KarpelesLab/qemuctl)** — QEMU
プロセスを管理するための Go ライブラリを12月29日に新規作成。

**[klbfw](https://github.com/KarpelesLab/klbfw)**、
**[atonline-flutter](https://github.com/KarpelesLab/atonline-flutter)**、
**[gobottle](https://github.com/BottleFmt/gobottle)** にも小さな
手入れ。
