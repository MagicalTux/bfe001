## algoc

**[algoc](https://github.com/KarpelesLab/algoc)** — リポジトリ初週
として26コミット。Rust / C 風の独自 DSL で書いたアルゴリズムを、
メモリ安全な実装として複数言語に出力するトランスパイラ。DSL
自体は強い型付け、明示的なメモリアノテーション、整数の拡張、
参照、配列とスライスを持つ — 同じ暗号／圧縮／コーデックの
アルゴリズムを Go / Rust / C / Swift と何度も書き直しては、
実装同士が食い違わないことを担保したい場面で効くもの。

汎用トランスパイラを目指しているわけではなく、性能が効く種類の
コードの言語ごとのボイラープレートを「一度書けば保てる」状態に
することが目的。
