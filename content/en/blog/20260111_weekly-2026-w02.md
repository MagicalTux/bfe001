## algoc

**[algoc](https://github.com/KarpelesLab/algoc)** — 26 commits on the
repo's first full week. A transpiler that takes an algorithm written
in a custom Rust/C-like DSL and emits optimized, memory-safe
implementations in multiple target languages. The DSL has strong
typing, explicit memory annotations, integer widening, references,
arrays and slices — the kind of guarantees you need when you're
writing the same cryptographic, compression, or codec algorithm over
and over in Go, Rust, C, and Swift and want the versions to actually
match.

The point isn't to be a general-purpose transpiler; it's to make the
per-language boilerplate for performance-critical code something you
maintain once.
