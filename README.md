# BFE001

Personal site of Mark Karpeles, live at [bfe001.net](https://www.bfe001.net).

Named after `$BFE001`, the CIA-A register on the Amiga that controlled the power LED and audio low-pass filter. `MOVE.b #$02,$BFE001` — one instruction that changed the physical state of the machine.

## What is this

A personal cartography of a mind that thinks in wireframes — from a ZX Spectrum in Dijon to the Amiga's memory-mapped hardware, and everything that wired itself in between.

## Tech

Static HTML/CSS/JS, hosted on GitHub Pages. No build step.
